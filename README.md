Простой логер CAN.
Нужна плата ESP32 (NodeMcu любая) и внешняя микросхема интерфейса SN65HVD232 (ноги подключения выбираются в коде, по умолчанию GPIO4 - Rx и GPIO5 - Tx). Для связи со смартфоном используется Serial Bluetooth Terminal.
Настройка идёт через команты из терминала (при старте скорость 500кбит/с):
 1) Настройка игнорирования ID в общем потоке (до 64 штук):
  
    D2 - все ID разрешены,
    
    D0=34 - выключить из приёма ID=0x34,
    
    D1=34 - включить приём ID=0x34,
    
    D9 - запрос включеных исключений), Можно выключить максимум 64 адресов, разделения стандартный или расширенный нету.

 3) Приём только одного единственного ID (если включено, игнорирует предыдущий список исключения):
  
    F0 - выключить фильтр,
    
    F1=34 - включить фильтр на ID=0x34,
    
    более приоритетная операция чем выключение ID из приёма, т.е. соответствующий ID пролезет на вывод если он выключен был опцией D1=xxxxx.

 5) Настройка скорости CAN:
  
    B500 - 500 кбит/с,
    
    B250 - 250 кбит/с,
    
    B125 - 125 кбит/с и т.п.).

 7) Вывод декодированый даный в читаемом виде (пишеться индивидуально по желанию в файле Decode.ino):
  
    V1 - включить вывод декодидированых значений и выключить вывод потока,
    
    V0 - выключить вывод декодированных и включить вывод потока.

Библиотеки используются все стандартные из набора Arduino IDE (v 2)
